Приложение Cargo Company — это настольная программа на C++/CLI (Windows Forms), предназначенная для моделирования работы грузовой компании. Система позволяет управлять тремя основными сущностями: тарифами, клиентами и заказами. Пользователь может добавлять, редактировать и удалять записи через графический интерфейс. Тарифы содержат базовую цену и стратегию скидок, клиенты — ФИО и паспортные данные (с проверкой уникальности), заказы связывают клиента с тарифом и объёмом перевозки с автоматическим расчётом стоимости. Редактирование данных реализовано через отдельное диалоговое окно с перегруженными конструкторами, визуально адаптирующееся под режим работы (тариф / клиент / заказ). Программа демонстрирует применение объектно-ориентированного подхода, паттерна «Стратегия» для расчёта скидок и базовых операций CRUD в Windows Forms интерфейсе.
